## Description

Multimodal Freight Sdn. Bhd. (MMF) is in the business of integrated logistics transportation in Peninsular Malaysia. This tender covers the supply, delivery, installation, testing, and commissioning of comprehensive IT systems across all MMF branches.

### Scope of Services

The Tenderer will be responsible for:
- Supply, delivery, installation, testing, commissioning of software
- Provide blueprint for each system
- Provide adequate training in operating the proposed software
- Provide resources for migrating MMF's existing data (softcopy and hardcopy) into IT System
- Provide support services for software and IT System application
