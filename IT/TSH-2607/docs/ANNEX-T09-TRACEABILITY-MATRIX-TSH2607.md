## TRACEABILITY MATRIX OVERVIEW

- Tender Requirements вҶ’ User Requirements
- User Requirements вҶ’ System Requirements  
- System Requirements вҶ’ Design Components
- Design Components вҶ’ Test Cases
- Test Cases вҶ’ Test Results
