## 1. COMMUNICATION STRATEGY

### 1.1 Purpose
This Communication Plan establishes the framework for effective information exchange among all stakeholders throughout the 67-month UCMS project lifecycle.

### 1.2 Communication Principles

| Principle | Application |
|-----------|-------------|
| Clarity | Use clear, jargon-free language |
| Timeliness | Deliver information when needed |
| Relevance | Share only pertinent information |
| Two-way | Encourage feedback and questions |
| Documented | Maintain records of key communications |
