## YEARLY CLAIM WITH AUDITOR TEMPLATE

### Auditor Working Template Structure

| Section | Description | Auditor Input |
|---------|-------------|---------------|
| A | Project Information | Auto-populated |
| B | Financial Summary | Verify from USPFS |
| C | Expenditure Verification | Check against invoices |
| D | Physical Progress Verification | Site visit/survey |
| E | Compliance Checklist | Verify requirements |
| F | Auditor Declaration | Sign-off |

### Required Documents for Yearly Claim

1. Audited Financial Statements
2. Auditor's Working Template
3. CEO Declaration
4. All Quarterly Claim Summaries
5. Supporting Invoices (consolidated)
6. Bank Statements
7. Progress Reports
