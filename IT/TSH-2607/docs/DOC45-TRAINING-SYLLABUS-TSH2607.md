## 1.0 TRAINING PROGRAM OVERVIEW

### 1.1 Training Objectives

By the end of this training program, participants will be able to:

- Navigate and use the UCMS application effectively
- Perform role-specific functions with confidence
- Understand USP claim workflows and procedures
- Troubleshoot common issues
- Adhere to security and compliance requirements

### 1.2 Target Audiences

| Audience | Role Description | Training Duration |
|----------|------------------|-------------------|
| Group A | MCMC Officers & Reviewers | 3 days |
| Group B | Service Provider Users | 2 days |
| Group C | System Administrators | 5 days |
| Group D | Public Users (Self-Service) | Online (2 hours) |

---

## 2.0 GROUP A: MCMC OFFICERS & REVIEWERS (3 Days)

### Day 1: UCMS Fundamentals

#### Session 1.1: Introduction to UCMS (2 hours)
- **Topics:**
  - USP program overview
  - UCMS purpose and benefits
  - System architecture (high-level)
  - User roles and responsibilities
- **Materials:**
  - Presentation: UCMS_Overview.pptx
  - Handout: USP_Program_Guide.pdf
- **Activities:**
  - Ice breaker and introductions
  - Quiz: USP Fundamentals

#### Session 1.2: System Navigation (2 hours)
- **Topics:**
  - Login and logout procedures
  - Dashboard overview
  - Menu navigation
  - Search functionality
  - User preferences
- **Materials:**
  - Quick Reference Card
  - Video: Navigation_Demo.mp4
- **Activities:**
  - Hands-on: Login and explore
  - Exercise: Find specific functions

#### Session 1.3: Applicant Management (3 hours)
- **Topics:**
  - Registering new applicants (individual)
  - Registering corporate applicants
  - Identity verification (MyIDENTITY)
  - Company verification (SSM)
  - Updating applicant information
  - Document upload and management
- **Materials:**
  - User Guide: Applicant_Management.pdf
  - Checklist: Applicant_Registration_Checklist.pdf
- **Activities:**
  - Lab: Register 5 test applicants
  - Lab: Verify applicant identity
  - Troubleshooting exercise

---

### Day 2: Claims Processing

#### Session 2.1: Claims Workflow (2 hours)
- **Topics:**
  - USP scheme overview
  - Eligibility criteria
  - Claim lifecycle
  - Status definitions
  - SLAs and timelines
- **Materials:**
  - Workflow Diagram: Claims_Process.pdf
  - Scheme Reference Guide
- **Activities:**
  - Case study: Eligibility scenarios
  - Group discussion: Common issues

#### Session 2.2: Creating and Submitting Claims (3 hours)
- **Topics:**
  - Creating new claims
  - Adding line items
  - Document attachment
  - Claim submission
  - Reference number tracking
- **Materials:**
  - Step-by-Step Guide: Submitting_Claims.pdf
  - Video Tutorial
- **Activities:**
  - Lab: Create 3 different claim types
  - Peer review exercise

#### Session 2.3: Claim Review and Approval (3 hours)
- **Topics:**
  - Review queue management
  - Eligibility verification
  - Document review
  - Approval process
  - Rejection and feedback
  - Requesting additional documents
- **Materials:**
  - Approval Authority Matrix
  - Rejection Reason Codes
- **Activities:**
  - Lab: Review and process 10 claims
  - Role-play: Difficult scenarios

---
