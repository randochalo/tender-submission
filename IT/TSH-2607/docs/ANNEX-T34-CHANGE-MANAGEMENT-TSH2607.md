## CHANGE MANAGEMENT FRAMEWORK

### Change Control Board (CCB)

| Role | Responsibility |
|------|----------------|
| Project Manager | Chair, overall coordination |
| MCMC Representative | Business approval |
| Solution Architect | Technical impact |
| QA Lead | Quality impact |
| Security Officer | Security review |

### Change Categories

| Category | Authority | Timeline |
|----------|-----------|----------|
| Emergency | CCB Chair | 24 hours |
| Standard | CCB | 1 week |
| Normal | CCB | 2 weeks |

### Change Process

```
Request ──► Impact Assessment ──► CCB Review ──► Approval ──► Implementation ──► Verification
```
