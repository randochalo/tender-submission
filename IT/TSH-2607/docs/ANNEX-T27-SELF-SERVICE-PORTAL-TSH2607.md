## SELF-SERVICE PORTAL FEATURES

### DUSP Self-Service Capabilities

| Feature | Description | Benefit |
|---------|-------------|---------|
| Claim Submission | Submit advance, quarterly, yearly claims online | 24/7 access |
| Document Upload | Upload supporting documents | Instant validation |
| Status Tracking | Real-time claim status | Transparency |
| Payment Tracking | View payment status and history | Accountability |
| Profile Management | Update company and contact info | Self-maintained |
| Reports Access | Download statements and reports | Convenience |

### Portal Security

- Azure AD SSO integration
- Role-based access
- Session timeout (30 minutes)
- IP whitelisting option
- Audit logging
