## 3. Scope of Work

### 3.1 Tenderer Responsibilities

The Tenderer will be responsible for performing the following tasks:

1. Supply, delivery, installation, testing, and commissioning of software for implementation of IT System for MMF
2. Provide blueprint for each system
3. Provide adequate training in operating the proposed software
4. Provide resources including labour and assist in migrating MMF's existing data (both softcopy and hardcopy) into IT System
5. Provide support services for software and IT System application

### 3.2 Training and Documentation

**Documentation:**
- Complete and comprehensive set of documentation including User Manual, Technical Design Manual, System Administration Manual and Database Design documentation

**Training Categories:**
- Administrator and IT team Training — Minimum officers as determined by MMF
- User Training — On-the-job training approach for proper operation

### 3.3 Support and Maintenance

- Provide support and maintenance services after "go live"
- Service Level Agreement (SLA) required
- 24/7 system availability
- Dedicated helpdesk service
- Warranty period coverage
